Cruise On TripAdvisor (CoTA)

I was the sole developer for a Java app that was an implementation of some of the major parts of Cruise Critic but presented on the main TripAdvisor page.

About the Projects

CoTA is part of the Micro-Service Architecture at TripAdvisor. It's written in Java and relys on Kubernettes for the server implementation. Any and all code changes were subject to the main site deployment schedule.

  • Date

    Aug 2022 - Aug 2023

Full Serarch

You could serach for destinations, Ports, and Ships. It pulled the parsed MetaData that was from the CruiseCritic Find A Cruise system.

Custom Deck Plans

Using AWS to host the image data, CoTA could dynamicly display Deck images and tie that to Deck info such as resturantes and stors.

Improved Reviews

I built changes to the ship review presentation to include data from the user submitted data fields.
